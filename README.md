# CadastroDeCarros

Um sistema web completo para **gerenciamento de cadastro de carros**, desenvolvido em **Java Spring Boot**, **Thymeleaf** e **H2 Database**.  
Estilo inspirado em sites de venda de veículos como OLX e WebMotors.

---

## 🚀 Tecnologias utilizadas

- **Java 25**
- **Spring Boot 4.0.3**
- **Spring Data JPA**
- **Thymeleaf**
- **H2 Database**
- **Flyway** (para migrations)
- **Bootstrap / CSS personalizado** para frontend
- **Lombok** para redução de boilerplate
- **.env** para armazenar as variaveis de ambiente

## 💻 Funcionalidades

- Adicionar, atualizar, listar e deletar carros
- Visualizar detalhes completos de cada carro
- Filtragem por **status** e **tipo de combustível**
- Sistema pronto para evoluir com galeria de imagens
- Página de detalhes com estilo OLX/WebMotors
